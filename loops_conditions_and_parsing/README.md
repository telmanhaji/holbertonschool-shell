This is readme file of the project "Shell, loops, conditions and parsing"
Here you can find everything about tasks and etc

0. 1-for_best_school - Bash script that displays Best School 10 times
1. 2-while_best_school - Bash script that displays Best School 10 times. Without using for and until.
2. 3-until_best_school - Bash script that displays Best School 10 times. Without using for and while.
3. 4-if_9_say_hi - Bash script that displays Best School 10 times, but for the 9th iteration, displays Best School and then Hi on a new line
4. 5-4_bad_luck_8_is_your_chance - Bash script that loops from 1 to 10 with 3 loop iterations.
5. 6-superstitious_numbers - script that displays numbers from 1 to 20 wth 3 loop iterations.
6. 7-clock - script that displays the time for 12 hours and 59 minutes.
7. 8-for_ls - script that displays: The content of the current directory, in a list format, where only the part of the name after the first dash is displayed.
8. 9-to_file_or_not_to_file - script that gives you information about the school file. Requirements:
	A. You must use if and, else (case is forbidden)
	B. Your Bash script should check if the file exists and print:
		- if the file exists: school file exists
		- if the file does not exist: school file does not exist
	C. If the file exists, print:
		- if the file is empty: school file is empty
		- if the file is not empty: school file is not empty
		- if the file is a regular file: school is a regular file
		- if the file is not a regular file: (nothing)
9.  10-fizzbuzz - script displays numbers from 1 to 100.Requirements:
	A. displays FizzBuzz when the number is multiple of 3 and 5
	B. displays Fizz when number is multiple of 3
	C. displays Buzz when the number is a multiple of 5
	D. otherwise displays the number
	E. in a list format
10. 11-read_and_cut - script that displays the content of the file /etc/passwd. Script should display:
	A. username
	B. user id
	C. Home directory path for the user.
	Requirements: Must use the while loop (for and until are forbidden)
11. 12-tell_the_story_of_passwd -  script that displays the content of the file /etc/passwd, using the while loop + IFS. Requirements: Must use the while loop (for and until are forbidden)
12. 13-lets_parse_apache_logs - script that displays the visitor IP along with the HTTP status code from the Apache log file.Requirements:
	A. Format: IP HTTP_CODE
		- in a list format
		- from example
	B. Must use awk
	C. Is not allowed to use while, for, until and cut
	D. Download and commit the apache-access.log file along with your answers files
13. 14-dig_the-data - script that groups visitors by IP and HTTP status code, and displays this data.Requirements:
	A. The exact format must be:
		- OCCURENCE_NUMBER IP HTTP_CODE
		- In list format
	B. Ordered from the greatest to the lowest number of occurrences
		- From  example
		- Must use awk
		- Is not allowed to use while, for, until and cut
