This is readme file of the project "Shell, loops, conditions and parsing"
Here you can find everything about tasks and etc

0. 1-for_best_school - Bash script that displays Best School 10 times
1. 2-while_best_school - Bash script that displays Best School 10 times. Without using for and until.
2. 3-until_best_school - Bash script that displays Best School 10 times. Without using for and while.
3. 4-if_9_say_hi - Bash script that displays Best School 10 times, but for the 9th iteration, displays Best School and then Hi on a new line
4. 5-4_bad_luck_8_is_your_chance - Bash script that loops from 1 to 10 with 3 loop iterations.
5. 6-superstitious_numbers - script that displays numbers from 1 to 20 wth 3 loop iterations.
6. 7-clock - script that displays the time for 12 hours and 59 minutes.
7. 8-for_ls - script that displays: The content of the current directory, in a list format, where only the part of the name after the first dash is displayed.
8. 10-fizzbuzz - script displays numbers from 1 to 100: A) displays FizzBuzz when the number is multiple of 3 and 5, B) displays Fizz when number is multiple of 3, C) displays Buzz when the number is a multiple of 5, D) otherwise displays the number, E) in a list format.
